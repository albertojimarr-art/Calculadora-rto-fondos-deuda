# Calculadora de Rendimiento

Calculadora para fondos de inversión de deuda utilizando:

- Base anual de 360 días
- Metodología simple de anualización

## Fórmula

Rendimiento anualizado =
((Precio Final / Precio Inicial) - 1) × (360 / días)

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
